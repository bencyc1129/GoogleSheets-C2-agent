GoogleSheets-C2-agent

The C2 agent that using google sheets to control victim host.

# Usage
## python script
1. `> pip install gdown google-api-python-client google-auth-httplib2 google-auth-oauthlib oauth2client`
2. `> python agent.py`

## executable binary
- windows <br>
`> agent.exe`
- linux <br>
`> ./agent`
